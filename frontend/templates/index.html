What is Cloud Native Maturity?

  • Cloud native maturity is a measure of just how ready to go cloud native your organisation is!

What does cloud native mean to me?

  • Cloud Native computing is a new way of developing software that makes use of cloud computing!
  • It uses technologies like containers, microservices and immutable infrastructure
  • Being Cloud native means you can build and run scalable applications in modern, dynamic environments
